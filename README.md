# Yetu.coders

Yetu.coders is an internal platform developed to manage project execution, team contribution, and financial distribution in a structured and transparent way.

## Overview

Yetu.coders centralizes how projects are executed within the team.

Instead of relying on scattered communication and subjective decisions, the platform provides a structured system where:
- Projects are clearly defined
- Tasks are assigned and tracked
- Contributions are measured objectively
- Financial distribution is calculated transparently

## Core Features

### Project Execution
- Create and manage projects with defined phases
- Assign responsibilities and deadlines
- Track progress in real time

### Task Management
- Structured task system with priorities and weights
- Status tracking (todo, in progress, review, done)
- Activity logs and history

### Contribution Tracking
- Measure individual participation based on real work
- Scoring system with weights, bonuses, and penalties
- Contribution percentage per project

### Financial Distribution
- Automatic split between company reserve and team pool
- Transparent payout calculation
- Project-level financial breakdown

### Company Reserve (Assets)
- Track company funds (património)
- Monitor income and expenses
- Categorize operational costs

### Team Structure
- Departments (Dev, QA, Design, etc.)
- Role-based access control
- Performance visibility

## Philosophy

Yetu.coders is built around three principles:

- Execution over assumptions  
- Data over opinion  
- Transparency over ambiguity
